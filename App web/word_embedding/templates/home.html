Word Embedding

Modelo Word2Vec

Word2Vec es un modelo de redes neuronales que aprende representaciones vectoriales de palabras a partir de grandes cantidades de texto sin etiquetar. Las representaciones capturan semántica y similitudes entre palabras.

Modelo Doc2Vec

Doc2Vec extiende el modelo Word2Vec para incluir representaciones vectoriales de documentos. Proporciona una forma de representar documentos enteros como vectores, conservando la información contextual.
